Porfolio Retro

Porfolio De Desarrollador Web

Mi portfolio web retro es un espacio que he creado para mostrar mi trabajo como desarrollador web.

En él, se pueden encontrar una selección de mis mejores proyectos, así como información detallada sobre mis habilidades y servicios.

Los visitantes podrán encontrar una sección de proyectos que incluye ejemplos de sitios web que he creado anteriormente. Estos proyectos varían en alcance y complejidad, pero todos ellos reflejan mi compromiso con la calidad y la atención al detalle.

Nuestra página web fue diseñada con el objetivo de ofrecerte la mejor experiencia posible en todos los dispositivos. Para lograr esto, comenzamos utilizando la herramienta de diseño Figma para crear los primeros bocetos e ideas. Una vez que tuvimos una idea clara de lo que quería lograr, utilicé las mejores prácticas de desarrollo web que hay actualmente, como HTML,PUG, CSS, SASS , JavaScript y diferentes librerías JavaScript para la zona de transiciones, para llevar nuestra visión a la vida.

De esta manera, logramos una página atractiva y fácil de usar en cualquier dispositivo, ya sea móvil, portátil, tablet, etc.
